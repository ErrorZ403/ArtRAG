## Setup

This project uses [Poetry](https://python-poetry.org/) for dependency management.

```shell
# Create Python environment
$ poetry install

# Install git pre-commit hooks
$ poetry shell
$ pre-commit install
```

## Running

```shell
# Run app.py
$ streamlit run app.py
```

# Running with Docker

```shell
# Run docker-compose
$ docker-compose up --build
```
